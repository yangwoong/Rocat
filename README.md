# Rocat
AI 에이전트 대시보드
물환경은 방제보다 예방이 중요합니다.
그러기 위해서는 매일관리가 필요하죠.
하지만, 현실적으로 매일관리는 어렵죠!! 이럴때 AI가 진가를 발휘합니다.

대화형 AI지휘에이전트를 통한 수질정보 관리와 필요시 로봇을 통한 개선으로 도심의 호수공원을 관리합니다.
사람과 AI 그리고 로봇이 함께 어려운 난관을 해결합니다.
